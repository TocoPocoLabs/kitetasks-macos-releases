### English

What's New in 1.0.17

New keyboard shortcuts
- ⌘N — Create a new task in the active list
- ⌘. — Toggle task completion
- ⌘1 / ⌘2 / ⌘3 — Jump to Today, Upcoming, or All tasks
- ⌘, — Open Settings
- ⌘↑ / ⌘↓ — Move selection within a column, or step across to the next column on the Kanban board
- ⌘F — Open search (replaces ⌘K)
- List and board views now share the same task ordering, so your selection moves through tasks exactly in the order
you see them
- All shortcuts are listed in Settings → Shortcuts

Localization updates
- Added full Korean (한국어) translation, available from the in-app language picker
- "Overdue", "Today", "Tomorrow", and "No Date" headers now appear in your selected language
- Polished Spanish and Portuguese wording in a few menus

Promo code redemption
- Got a promo code? Tap "Have a promo code?" in the paywall, purchase screen, or Settings → License to redeem it via
the Mac App Store

Other improvements
- Occasional prompt to rate KiteTasks after you've been using a paid plan — never shown during the free trial
- Various small fixes and polish

### 🇯🇵 Japanese

バージョン 1.0.17 の新機能

新しいキーボードショートカット
- ⌘N — アクティブなリストに新しいタスクを作成
- ⌘. — タスクの完了/未完了を切り替え
- ⌘1 / ⌘2 / ⌘3 — 「今日」「今後の予定」「すべてのタスク」にジャンプ
- ⌘, — 設定を開く
- ⌘↑ / ⌘↓ — 列内で選択を移動、またはかんばんボードで次の列へ移動
- ⌘F — 検索を開く (⌘K から変更)
- リストビューとボードビューでタスクの順序が共通化され、見た目通りの順番で選択がスムーズに移動するようになりました。
- すべてのショートカットは「設定 > ショートカット」で確認できます。

ローカリゼーションの更新
- 韓国語 (한국어) の完全な翻訳を追加しました。アプリ内の言語選択から利用可能です。
- 「期限切れ」「今日」「明日」「日付なし」のヘッダーが選択した言語で正しく表示されるようになりました。
- スペイン語とポルトガル語の一部のメニューテキストを改善しました。

プロモコードの利用
- プロモコードをお持ちですか？ ペイウォール、購入画面、または「設定 > ライセンス」にある「プロモコードをお持ちですか？」をクリックして、Mac App Store 経由で適用できます。

その他の改善
- 有料プランをご利用中の方に向けて、KiteTasks の評価をお願いするポップアップを時折表示するようになりました（無料トライアル中には表示されません）。
- 細かなバグ修正とアプリの動作改善。

### 🇰🇷 South Korean

버전 1.0.17의 새로운 기능

새로운 키보드 단축키
- ⌘N — 현재 활성화된 목록에 새 할 일 만들기
- ⌘. — 할 일 완료 상태 전환
- ⌘1 / ⌘2 / ⌘3 — '오늘', '다가오는 일정', '모든 할 일'로 이동
- ⌘, — 설정 열기
- ⌘↑ / ⌘↓ — 열 내에서 선택 항목 이동, 또는 칸반 보드에서 다음 열로 이동
- ⌘F — 검색 열기 (⌘K에서 변경됨)
- 리스트 뷰와 보드 뷰가 이제 동일한 할 일 정렬 순서를 공유하므로, 눈에 보이는 순서 그대로 선택 항목이 자연스럽게 이동합니다.
- 모든 단축키는 설정 > 단축키에서 확인할 수 있습니다.

현지화 업데이트
- 완벽한 한국어 번역이 추가되었습니다 (앱 내 언어 설정에서 선택 가능).
- '기한 지남', '오늘', '내일', '날짜 없음' 헤더가 이제 선택한 언어로 올바르게 표시됩니다.
- 일부 스페인어 및 포르투갈어 메뉴 텍스트를 다듬었습니다.

프로모션 코드 사용
- 프로모션 코드가 있으신가요? 결제 화면, 구매 화면 또는 설정 > 라이선스에서 '프로모션 코드가 있으신가요?'를 클릭하여 Mac App Store를 통해 코드를 등록할 수 있습니다.

기타 개선 사항
- 유료 플랜을 사용 중인 분들을 위해 간헐적으로 KiteTasks 평가 요청이 표시됩니다 (무료 체험 기간 중에는 절대 표시되지 않습니다).
- 기타 다양한 소규모 버그 수정 및 완성도 향상.
