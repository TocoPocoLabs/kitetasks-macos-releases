Version 1.0.14

##  English
Link Support
- Tasks with links now display them as clickable chips — tap to open in your browser
- Paste a URL when creating a task and KiteTasks will automatically fetch the page title and use it as
the task name
- A link icon now appears on task cards so you can quickly spot tasks with links attached

---

## Spanish

Soporte para enlaces
- Las tareas con enlaces ahora los muestran como etiquetas ("chips") en las que puedes hacer clic para abrirlos directamente en tu navegador.
- Pega una URL al crear una tarea y KiteTasks obtendrá automáticamente el título de la página para usarlo como el nombre de la tarea.
- Ahora aparece un icono de enlace en las tarjetas de las tareas para que puedas identificar rápidamente cuáles tienen enlaces adjuntos.

### Portuguese

Suporte a links
- Tarefas com links agora os exibem como etiquetas (chips) clicáveis — clique para abri-los no seu navegador.
- Cole um URL ao criar uma tarefa e o KiteTasks buscará automaticamente o título da página para usá-lo como o nome da tarefa.
- Um ícone de link agora aparece nos cartões de tarefas para que você possa identificar rapidamente aquelas que contêm links anexados.

### Japanese

リンク対応
- リンクを含むタスクに、クリック可能なチップが表示されるようになりました。クリックするとブラウザで開きます。
- タスク作成時にURLをペーストすると、KiteTasksが自動的にページのタイトルを取得し、タスク名として設定します。
- タスクカードにリンクアイコンが表示されるようになり、リンクが添付されているタスクを一目で確認できるようになりました。
